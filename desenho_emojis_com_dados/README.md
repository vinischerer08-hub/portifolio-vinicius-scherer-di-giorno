🎨 Processamento Multimodal: Imagens, Emojis e Frequências
📝 Descrição do Projeto
Este projeto explora a manipulação de estruturas de dados complexas (dicionários aninhados, listas e tuplas) para o processamento de diferentes tipos de mídia via Python. O foco principal é demonstrar como algoritmos de iteração em múltiplos níveis podem ser aplicados para transformar dados brutos em representações visuais ou musicais.

Desenvolvido para fins acadêmicos, o sistema aborda três frentes principais:

Manipulação de Pixels RGB: Aplicação de filtros de brilho em grades de cores para criação e sombreamento de emojis.

Transposição de Matrizes Sonoras: Reorganização de frequências musicais utilizando lógica de álgebra linear (matriz transposta).

Gestão de Playlists de Imagens: Processamento em lote de coleções de dados visuais com aplicação de filtros de redução de luminosidade.

🚀 Tecnologias Utilizadas
Linguagem: Python 3.x

Bibliotecas: Matplotlib (Visualização de dados e renderização de imagens)

Ferramentas: Google Colab, VS Code

📊 Funcionalidades e Aprendizados
O projeto demonstra o domínio de lógica de programação aplicada a ciência de dados e multimídia:

Filtro de Sombreamento: Implementação de lógica condicional para identificar cores específicas (ex: Amarelo) e reduzir seus valores RGB em 50%, simulando um efeito de sombra.

Processamento Multidimensional: Uso de loops aninhados (até 3 níveis) para percorrer dicionários e listas complexas sem perder a integridade dos dados.

Transformação de Dados: Algoritmo de transposição manual de matrizes, essencial para converter estruturas de dados de áudio ou transformar a orientação de imagens.

Gerenciamento de Memória: Uso de métodos como .pop() e .insert() para manipulação dinâmica de coleções de objetos.
