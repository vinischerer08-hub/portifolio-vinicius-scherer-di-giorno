💻 Lógica de Programação: Traduzindo Algoritmos para Python
📝 Descrição do Projeto
Este repositório contém uma série de algoritmos práticos desenvolvidos para consolidar conceitos fundamentais de lógica de programação utilizando a linguagem Python. O projeto é dividido em quatro módulos principais (etapas), cada um focado em resolver problemas do cotidiano através de estruturas de controle, repetição e manipulação de dados.

O objetivo central é demonstrar a aplicação de:

Estruturas Condicionais: Tomada de decisão baseada em valores e limites.

Laços de Repetição: Processamento de múltiplos dados e iterações definidas.

Operações Matemáticas: Cálculos de descontos, médias aritméticas, juros compostos e acumuladores.

Entrada/Saída de Dados: Interação dinâmica com o usuário via terminal.

🚀 Estrutura do Sistema
O projeto está organizado em quatro funções independentes:

Processamento de Vendas: Sistema de checkout que calcula o total de compras, valida entradas negativas e aplica descontos progressivos (5% ou 10%) baseados no valor total.

Análise Climática: Monitoramento semanal de temperatura com cálculo de média e emissão de alertas de segurança para condições térmicas extremas.

Sistema de Notas: Gestão acadêmica para cálculo de médias de alunos, classificando-os automaticamente em "Aprovado", "Recuperação" ou "Reprovado".

Simulador de Poupança: Ferramenta financeira que projeta o rendimento de investimentos com aportes mensais e juros compostos, incluindo monitoramento de metas financeiras.

🛠️ Tecnologias Utilizadas
Linguagem: Python 3.x

Ambiente de Desenvolvimento: Google Colab / VS Code

📊 Conceitos Aplicados
Acumuladores e Contadores: Utilizados para somar totais de vendas e contar dias de calor extremo.

Validação de Dados: Filtros para impedir que preços ou quantidades inválidas corrompam os cálculos.

Flags (Sinalizadores): Uso de variáveis booleanas para detecção de alertas críticos de clima.

Interpolação de Strings: Formatação de mensagens de saída para melhor experiência do usuário.
