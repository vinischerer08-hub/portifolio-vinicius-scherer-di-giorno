💻 Sistema de Auditoria de Vendas e Simulações de Negócios
📝 Descrição do Projeto
Este repositório contém um conjunto de algoritmos desenvolvidos para automação de processos acadêmicos e empresariais. O projeto central é um Sistema de Auditoria de Vendas, projetado para identificar discrepâncias financeiras, gerenciar limites de segurança e automatizar a tomada de decisão em cenários de quarentena de dados.

Além do núcleo financeiro, o projeto inclui módulos de processamento de Visão Computacional para carros autônomos, análise de Microclima Urbano e simuladores de Pegada de Carbono, demonstrando a aplicação de lógica de programação em problemas complexos do mundo real.

🚀 Tecnologias e Conceitos Aplicados
Linguagem: Python 3.10

Paradismas: Programação Funcional e Estruturada.

Bibliotecas: Matplotlib (Visualização), FastAPI (Interface de API), Pydantic (Validação de dados).

Conceitos Chave: * Lógica de matrizes (Pixels RGB e Visão Computacional).

Estruturas de repetição aninhadas e controle de estado (Flags).

Cálculos financeiros e atuariais (Juros compostos e lucratividade).

📊 Módulos Principais
O arquivo está organizado em etapas evolutivas de aprendizado e aplicação:

Auditoria de Vendas: Sistema que monitora médias de vendas, detecta discrepâncias (5x acima da média) e permite ao gestor redefinir limites globais de segurança.

Visão Computacional: Algoritmo que simula a leitura de sensores de um carro autônomo, identificando objetos por brilho de pixel e acionando freios de emergência.

Green Tech: Simulador de emissões de CO2 baseado na matriz energética (Solar, Eólica, Carvão ou Gás) para data centers.

Microclima Urbano: Análise de índices de qualidade do ar (IQA) e conforto térmico em pontos específicos de São Paulo.

API de Rotina: Implementação de um backend utilizando FastAPI para gerenciamento de tarefas (CRUD).
